## Descrição

Este repositório contém um **projeto de banco de dados para um sistema de vendas**, desenvolvido como **fundamento prático de aprendizagem em SQL**.

O projeto serviu como base de estudo antes da construção de um sistema acadêmico mais completo, permitindo praticar **modelagem de dados, criação de scripts SQL** e **consultas** em um cenário realista.

---

## Objetivo do Projeto

• Praticar **modelagem conceitual e lógica**
• Criar um **banco de dados relacional**
• Desenvolver **scripts SQL completos**
• Exercitar **consultas SQL para análise de dados**
• Consolidar fundamentos de **MySQL**
